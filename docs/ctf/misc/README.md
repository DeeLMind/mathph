## 什么是CTF
CTF（Capture The Flag）是一个```游戏```，是专门给网络安全人员玩的一个```游戏```，所以你要知道你还不是安全人员之前请先想办法学习一下安全技术，先变成一名网络安全人员，而不是只知道```玩游戏```。
