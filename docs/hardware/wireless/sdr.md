# SDR无线

## 什么是SDR

SDR的意思是软件无线电，Software Defined Radio，直接用软件控制硬件设置，无需频繁更换硬件设置。

## SDR分类
* [HackRF](https://greatscottgadgets.com/hackrf/one/)
* [RTL-SDR](https://www.rtl-sdr.com/)
* [BladeRF](https://www.nuand.com/bladerf-1/)

* 收发信号
* 分析信号
