# ATT&CK

## [MITRE ATT&CK](https://attack.mitre.org/)

MITRE ATT&CK模型它是根据真实的观察数据来描述和分类对抗行为。MITRE ATT&CK将已知攻击者行为转换为结构化列表矩阵，将这些已知的行为汇总成战术和技术，因此对于各种进攻性和防御性度量、表示和其他机制都非常有用。
