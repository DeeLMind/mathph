# 爆破弱口令

## 什么是弱口令
弱口令就是比较简单，容易被猜测出的密码，口令等等。例如：

`
123456789
a123456
123456
a123456789
1234567890
woaini1314
qwert
`

<DocsAD/>

## 什么是爆破字典
字典其实就是一个大的密码集合，骇客可以用字典去爆破猜解对方的密码。

## 什么是爆破弱口令
通过已有的字典爆破，弱口令爆破，进而访问受害者的敏感资源。

[Burpsuite](../../tools/burpsuite/README.md)可以满足大部分要求
