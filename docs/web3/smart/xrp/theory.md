# 合约原理

[XRP](https://xrpl.org/docs/) 是一种由 Ripple Labs 创建的加密货币，专为快速、低成本的跨境支付和资金转移而设计。它在 Ripple 网络中扮演了中介货币的角色，主要用于金融机构和银行之间的国际结算和支付。XRP 的主要目标是解决传统银行跨境支付中的延迟和高费用问题，提供一个更高效的替代方案。
