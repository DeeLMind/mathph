## 拜占庭算法

拜占庭容错算法（Byzantine Fault Tolerance，BFT）旨在解决这个问题，确保系统在存在恶意节点或错误节点的情况下仍能达成一致。常见的拜占庭容错算法有PBFT（Practical Byzantine Fault Tolerance）、Tendermint和Algorand等。
