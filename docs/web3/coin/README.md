# 虚拟货币

## 什么是虚拟货币
密码货币（英语：Cryptocurrency，又译密码学货币、加密货币）是一种数字形式的货币，它没有实体形态，完全基于网络和电子技术。虚拟货币通过加密技术来确保交易的安全性和匿名性，通常由区块链或其他分布式账本技术进行管理和记录。
