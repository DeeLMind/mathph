## OpenVPN

* ubuntu 低版本

```shell
wget https://git.io/vpn -O openvpn-install.sh
chmod 777 ./openvpn-install.sh
sudo ./openvpn-install.sh
```

* ubuntu 高版本

```shell
wget -O openvpn-install.sh https://get.vpnsetup.net/ovpn
chmod 777 ./openvpn-install.sh
sudo ./openvpn-install.sh
```
