# 命令行

## cargo

* **`新建工程`**：cargo new cmd
* **`运行工程`**：cargo run
* **`运行检查`**：cargo check
* **`编译工程`**：cargo build (Debug)
* **`编译工程`**：cargo build --release
* **`清理缓存`**：cargo clean
* **`第三方库`**：cargo add lib_name

## rustup

* **`更新Rust`**：rustup update
* **`卸载Rust`**：rustup self uninstall
* **`安装目录`**：rustup show home
* **`查看工具`**：rustup show
* **`安装工具`**：rustup install xxx

## rustc

* **`运行脚本`**：rustc main.rs
